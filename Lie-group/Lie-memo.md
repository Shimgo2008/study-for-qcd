# Lie群
わがんない！！！

というか、群論をちゃんと勉強しないと理解できないため、直感的な理解ができるLie括弧とLie代数についてまとめる。

Lie群は最終成果物と言うような理解に留めておく。
ざっくばらんに言うと、Lie群とはとある「無限小変換」の集合を「指数写像」でつみ上げたもの。

この無限小変換の集合がLie代数であり、無限小変換同士の順序依存性を表すのがLie括弧である。
```math
\exp: \mathfrak{g} \to G
```
ここで、$\mathfrak{g}$はLie代数、$G$はLie群を表す。

どうして無限小を積み重ねる、という操作なのに、積分ではなく指数写像なのか？という疑問が湧くが、これは無限小変換を有限変換に拡張する方法として指数写像が用いられるためである。

実際に無限小変換$X$を指数写像に乗っけると、
```math
\exp(X) = \sum_{n=0}^{\infty} \frac{X^n}{n!}
```


# Lie括弧
```math
[X, Y] = XY - YX
```
Lie括弧(Lie bracket)とは、次に紹介するLie代数という一つのベクトル空間に定義される二項演算である。

これにおける$X$, $Y$は行列。
イメージとしては、行列の掛け算は非可換であることを利用して、2つの行列の「差」を取る。
これは物理的には二つの作用の順番を入れ替えた時の差を表している。

e.g.量子力学
```math
[\hat{x}, \hat{p}] = i\hbar
```
ここで、$\hat{x}$は位置演算子、$\hat{p}$は運動量演算子、$\hbar$はプランク定数を$2\pi$で割ったもの。

# Lie代数
無限小変換で表される行列の集合のこと。

一般にLie代数は$\mathfrak{g}$で表され、Lie括弧を用いて次の性質を満たす。
1. 閉包性: 任意の$X, Y \in \mathfrak{g}$に対して、$[X, Y] \in \mathfrak{g}$である。

2. 反対称性: 任意の$X, Y \in \mathfrak{g}$に対して、$[X, Y] = -[Y, X]$である。

3. ヤコビ恒等式: 任意の$X, Y, Z \in \mathfrak{g}$に対して、$[X, [Y, Z]] + [Y, [Z, X]] + [Z, [X, Y]] = 0$である。

Jacobi恒等式は、複数の作用の順序依存性が矛盾しないことを保証する式である。
Lie代数は、Lie群の無限小変換を記述するために用いられ、物理学や微分幾何学などで重要な役割を果たす。

この無限小変換を指数写像を用いてつみ上げることで、Lie群を得ることができる。

指数写像とは、無限小変換を有限変換に拡張する方法であり、これを用いて、Lie代数からLie群への対応を構築する。
```math
\exp: \mathfrak{g} \to G
```
