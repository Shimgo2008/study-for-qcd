## SU(n)をLie群として定義しようの会
とりあえず、SU(n)とは何ぞや？ってとこから。
当たり前だけどいきなり群から行くのは無理があるので、Lie群の定義から。
まず
```math
\mathcal{A} = M_n(\mathbb{C})
```
ここで、$M_n(\mathbb{C})$は$n\times n$の複素行列全体の集合のこと。
って言うめちゃくちゃ大きな集合を考える。
この中で、次の二つの条件を満たす行列全体の集合を考える。
1. 反エルミート行列であること。
2. 単位行列に比例する行列を除くこと。

これを数式で表すと、
1. 反エルミート条件: $X^{\dagger} = -X$
2. トレースゼロ条件: $\mathrm{Tr}(X) = 0$
となる。
このとき、Lie代数の元 $X$ は、反エルミート条件 $X^{\dagger} = -X$ を満たす必要がある。

一方、物理ではこの無限小変換 $X$ を、いくつかの基本的な「回転の軸」（生成子 $T^a$）の組み合わせとして解釈したい。この考え方は、ベクトルを基底ベクトルで展開するのと同じである。すなわち、$X$ は $T^a$ の線形結合で書けるはずである。
```math
X = \sum_{a=1}^{n^2-1} c_a T^a
```
ここで、係数 $c_a$ がそれぞれの「軸」に対する「回転の量」に対応する。

さらに物理的な要請として、「回転の軸」である生成子 $T^a$ は、観測可能量と結びつけやすい**エルミート行列 ($T^{a\dagger}=T^a$)** であることが望ましい。

ここで問題が生じる。エルミート行列 $T^a$ の線形結合が、反エルミート行列 $X$ になるためには、係数 $c_a$ はどのような数であるべきだろうか？

これを解決するのが**虚数単位**と**実数のパラメータ $\theta_a$** である。係数 $c_a$ を純虚数 $c_a = -i\theta_a$ （ここで $\theta_a$ は実数）と置くことで、すべての条件が満たされる。

したがって、$X$ は以下のように表現される。
```math
X = -i \sum_{a=1}^{\dim \mathfrak{su}(n)} \theta_a T^a
```
この表現を用いると、物理的に意味のある「回転量」である実数 $\theta_a$ が、数式の中に自然に導入される。この定義で $X^\dagger = -X$ が成立することは（前回の回答のように）簡単に確認できる。



この二つの条件を満たす行列全体の集合を以下のように書き記すことができる。
```math
\mathfrak{su}(n) = \{ X \in M_{n}(\mathbb{C}) | X^{\dagger} = -X, \mathrm{Tr}(X) = 0 \}
```
e.g. $n=2$のとき、$\mathfrak{su}(2)$の元 $X$ は、3つの独立なパラメータ（回転量）$\theta_j$ と、3つの基底（回転の軸）であるパウリ行列 $\sigma_j$ を用いて以下のように表現できる。
```math
X = -i \left( \theta_1 \frac{\sigma_1}{2} + \theta_2 \frac{\sigma_2}{2} + \theta_3 \frac{\sigma_3}{2} \right)
```
ここで、生成子としてパウリ行列そのものではなく、$\frac{\sigma_j}{2}$ を用いるのは、これがスピン角運動量演算子の交換関係を正しく満たすためである。
このように、$\mathfrak{su}(2)$は3つの独立なパラメータで記述されるため、その次元は3である。
```math
\dim \mathfrak{su}(2) = 2^2 - 1 = 3
```

なぜ、$\frac{\sigma_j}{2}$なのかというと、量子力学におけるスピン角運動量演算子がパウリ行列の半分で表されるからである。

これらの行列は3つの独立なパラメータ$a_1, a_2, a_3$で表されるため、$\mathfrak{su}(2)$の次元は3であると言える。

これは
```math
\dim \mathfrak{su}(n) = n^2 - 1
```
と一般化できる。

これで、SU(n)のLie代数$\mathfrak{su}(n)$が定義できた。
最後に、これを指数写像に乗っけてLie群SU(n)を定義する。
```math
\exp: X \to SU(n)| X \in \mathfrak{su}(n)
```
これがSU(n)の定義。

# 問
幾つの独立な順序依存の操作があるのか？ $\fallingdotseq$quarkの色がもたらす生成子の数はいくつ必要か？

# 制約条件
SU(n)の生成子はLie代数$\mathfrak{su}(n)$の基底で与えられる。
Lie代数$\mathfrak{su}(n)$の次元は$n^2 - 1$であるため、SU(n)の生成子の数も$n^2 - 1$である。
したがって、quarkの色に対応する生成子の数は次のようになる。
```math
\text{生成子の数} = n^2 - 1
```
例えば、$n=3$の場合、生成子の数は
```math
\text{生成子の数} = 3^2 - 1 = 8
```
となる。

# 結論
SU(3)の生成子の数は8であり、これは実験により予測されていた、gluonの種類の数と一致する。
したがって、SU(3)がquarkの色の対称性を記述するのに適していることがわかる。

# 結論
SU(3)の生成子の数は $3^2-1=8$ であり、これは強い相互作用を媒介する粒子（グルーオン）が8種類あるという実験事実と一致する。

歴史的には、SU(3)対称性は最初、u,d,sの3つのフレーバーを分類する枠組み（フレーバーSU(3)）として導入された。しかし、このクォークモデルでは、例えば$\Delta^{++}$粒子が同じスピンを持つ3つのuクォークで構成されることになり、フェルミ粒子のパウリの排他原理と矛盾するという問題が生じた。
この困難を解決するために、クォークにはフレーバーとは別に「カラー（色）」と呼ばれる新しい3つの自由度（赤、緑、青）が存在すると提唱された。この**カラー自由度に対するSU(3)対称性**こそが、現在のQCDの基礎となるゲージ理論である。

したがって、この3種類の自由度はflavorではなく、色であると解釈されるようになった。

# おまけ
SU(3)の生成子は8つであり、これらはGell-Mann行列として知られている。
Gell-Mann行列はSU(3)のLie代数$\mathfrak{su}(3)$の基底を形成し、次のように表される。
```math
\lambda_1 = \begin{pmatrix} 0 & 1 & 0 \\ 1 & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix}, \quad
\lambda_2 = \begin{pmatrix} 0 & -i & 0 \\ i & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix}, \quad
\lambda_3 = \begin{pmatrix} 1 & 0 & 0 \\ 0 & -1 & 0 \\ 0 & 0 & 0 \end{pmatrix}, \quad\\
\lambda_4 = \begin{pmatrix} 0 & 0 & 1 \\ 0 & 0 & 0 \\ 1 & 0 & 0 \end{pmatrix}, \quad
\lambda_5 = \begin{pmatrix} 0 & 0 & -i \\ 0 & 0 & 0 \\ i & 0 & 0 \end{pmatrix}, \quad
\lambda_6 = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & 1 \\ 0 & 1 & 0 \end{pmatrix}, \quad\\
\lambda_7 = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & -i \\ 0 & i & 0 \end{pmatrix}, \quad
\lambda_8 = \frac{1}{\sqrt{3}} \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & -2 \end{pmatrix}
```
これらの行列は、quarkの色の変換を記述するために使用される。
