## 推奨する読み順 (Roadmap)

理論の依存関係に基づき、以下の順序で読むことを想定しています。

1.  **EL-equation (`EL-equation/`)**
    *   汎関数微分の定義（Gâteau微分）を確認し、古典場の理論の基礎となるオイラー=ラグランジュ方程式を導出。

2.  **Relativity (`Relativity/`)**
    *   幾何学的な距離 $ds^2$ の定義から出発し、相対論的な運動量とエネルギーの関係 $E^2 = p^2c^2 + m^2c^4$ を導出。


3.  **KG-equation (`KG-equation/`)**
    *   シュレディンガー方程式を相対論的に拡張し、スカラー場の波動方程式を導出。

4.  **Dirac-equation (`Dirac-equation/`)**
    *   KG方程式の負の確率密度問題を解決するために、Clifford代数 $Cl(4,0)$ を導入。
    *   パウリ行列を用いた行列表現を構成し、ミンコフスキー時空へ接続してディラック方程式を導出。

5.  **Lie-group (`Lie-group/`)**
    *   内部自由度としての「色（Color）」を記述する SU(3) 群と、その生成子（Gell-Mann行列）についてのメモ。

6.  **QCD-Lagrangian (`QCD-Lagrangian/`)**
    *   ディラック場に SU(3) 局所ゲージ対称性を要請。
    *   共変微分 $D_\mu$ とゲージ場（グルーオン） $A_\mu$ の必然的な出現、および非可換性による場の強さ $F_{\mu\nu}$ の導出。

7. **Wilson-loop (`Wilson-loop/`)**
    *   格子ゲージ理論における Wilson ループの定義と、その物理的意味についてのメモ。
